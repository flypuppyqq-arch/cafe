# 사내 카페 예약 시스템

사내 직원들을 위한 카페 예약 랜딩페이지입니다.

## 주요 기능

- 카페 소개 (사진, 위치, 운영시간, 메뉴)
- 온라인 예약 시스템
- Google Sheets 연동으로 예약 데이터 관리

## 기술 스택

- **프론트엔드**: HTML, CSS, JavaScript
- **백엔드**: Google Apps Script
- **데이터베이스**: Google Sheets

## 파일 구조

```
cafe/
├── index.html          # 메인 웹페이지
├── style.css          # 스타일시트
├── script.js          # JavaScript 로직
├── Code.gs            # Google Apps Script 코드
├── 구현_설명서.md      # 상세한 구현 가이드
└── README.md          # 프로젝트 소개
```

## 빠른 시작

1. **구현_설명서.md** 파일을 열어 단계별 설치 가이드를 따라하세요
2. 비개발자도 쉽게 따라할 수 있도록 상세하게 작성되어 있습니다

## 설치 요약

1. Google Sheets 생성
2. Google Apps Script 설정 및 배포
3. `script.js`에 Apps Script URL 입력
4. 웹사이트 배포 (로컬 또는 GitHub Pages)

## 사용자 정의

`index.html`과 `style.css`를 수정하여:
- 카페 정보 (이름, 위치, 운영시간)
- 메뉴 항목
- 색상 테마
- 카페 사진
